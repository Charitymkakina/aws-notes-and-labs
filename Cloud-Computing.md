# Cloud Computing
It is the on-demand delivery  of IT resources over the internet with pay-as-you-go pricing.
With Cloud Computing, you only pay for what you are using.
You can provision exactly the right size and type of computing resources you need.
You can access as many resources as you need almost instantly.
Cloud computing offers a simple way to access servers, storage, databases, and a set of application services.

# Cloud Deployment Models
1. `Private Cloud` - Cloud services used by a single organization and not exposed to the public. It offers complete control and security for sensitive applications and may meet specific business needs. 
2. `Public Cloud` - Cloud resources owned and operated by a third-party cloud service provider and delivered over the internet.
3. `Hybrid Cloud` - A combination of the public cloud and private cloud. You have control over sensitive assets in your private infrastructure and still have the flexibility and cost-effectiveness of the public cloud.

# Characteristics of Cloud Computing
1. On-demand self-service- Users can provision resources and use them without human interaction from the service provider.
2. Broad-Network access- Resources available over the network can be accessed by diverse client platforms.
3. Multi-tenancy and resource pooling- Multiple customers can share the same infrastructure and applications with security and privacy.
4. Rapid elasticity and scalability- Customers can automatically and quickly acquire and dispose of resources when needed. Also, customers can quickly and easily scale base on demand.
5. Measured service- Usage is measured, and users pay correctly for what they have used.

# Advantages of Cloud Computing
1. Trade capital expenditure (CAPEX) for operational expenditure (OPEX). Customers won't spend money buying servers; instead, they just rent.
2. Benefit massively from economies of scale. Prices are reduced as AWS is more efficient due to its large scale.
3. Stop guessing capacity- Instead of buying servers with expected usage of traffic, cloud computing offers scalability based on the actual measured usage
4. Increased speed and agility.
5. Stop spending money running and maintaining data centers.
6. Go global in minutes thanks to leveraging AWS global infrastructure.
