# S3
is an object storage service provided by Amazon Web Services that allows you to store and retrieve any amount of data from anywhere on the web.

# Components

1. Buckets
Containers used to store objects
Must have a globally unique name
Defined at the region level

2. Objects
Files stored in S3 (e.g., images, videos, documents)
Consists of:
Key (name of the object)
Value (data/content)
Metadata

4. Key
Unique identifier for an object within a bucket
Example: folder1/image.jpg

# S3 Storage Classes
1. S3 Standard: General-purpose storage, high availability
2. S3 Intelligent-Tiering: Automatically moves data based on usage
3. S3 Standard-IA: Infrequent access, lower cost
4. S3 One Zone-IA: Stored in one AZ (less resilient, cheaper)
5. S3 Glacier: Long-term archival (minutes to hours retrieval)
6. S3 Glacier Deep Archive: Lowest cost, long-term storage (hours retrieval)
