# Cloud Service Model
1. Infrastructure as a service
2. Platform as a service
3. Software as a service

# Infrastructure as a service (IaaS)
Provides building blocks for cloud IT. Provides networking, computers and data storage space. Offering the highest level of flexibility
and easy parallel migration with traditional on-premises IT.
An example of IaaS is the EC2 offered by Amazon

# Platform as a service (Paas)
Removes the need for an organization to manage the underlying infrastructure. 
While allowing the organization to focus on the deployment and management of its applications. An example of PaaS is Elastic Beanstalk.

# Software as a Service (SaaS)
Complete product run and managed by the service provider. An example of SaaS is Rekognition machine learning model on AWS.


# Pricing of the cloud
AWS has 3 pricing fundamentals. That follows the pay-as-you-go model.
1. Compute- Pay for the compute time
2. Storage- Pay for the exact amount of data stored in the cloud.
3. Networking- Pay for data transfer out of the cloud. Data transfer in the cloud is free.
